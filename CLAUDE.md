# Project Guidelines

## Blog Posts

- Write in an **agent-agnostic** style — avoid referencing Claude, ChatGPT, or any specific AI by name. Use "agent", "AI agent", or "your agent" instead.
- When referencing the Alby Bitcoin Payments skill, use the install command: `npx skills add getAlby/payments-skill`
- Write "bitcoin" and "lightning network" in lowercase, same as "the internet" — only capitalise at the start of a sentence or in a title.
